**Design Movement:** Dark Terminal / Sports Analytics Command Center

**Core Principles:**
- Pure black canvas with surgical precision typography
- Data-first hierarchy: numbers command attention, labels recede
- Monochromatic depth: layers of dark grays create spatial separation
- Neon accent sparingly used only for actionable picks

**Color Philosophy:**
- Background: #080810 (near-black with blue undertone)
- Card surface: #0f0f1a
- Border: rgba(255,255,255,0.06)
- Primary accent: #6c63ff (violet-purple for CTAs)
- Pick accent: #22c55e (green for model picks)
- Muted text: #6b7280

**Layout Paradigm:**
- Full-width single column feed of game cards
- Centered title bar, sticky header
- Cards separated by date/time groups
- Three-column data grid inside each card (Books | Model Line | Model O/U)

**Signature Elements:**
- Uppercase tracked letter headers (letter-spacing: 0.15em)
- Pill-shaped model line badges with subtle glow
- Edge points in green with triangle indicator

**Interaction Philosophy:**
- Hover states reveal share/expand actions on card right edge
- Smooth tab transitions on login form
- Minimal animation, data-focused

**Animation:**
- Card entrance: fade-in with 20px upward slide, staggered
- Tab switch: smooth cross-fade
- Button hover: subtle brightness increase

**Typography System:**
- Display: "Rajdhani" or "Barlow Condensed" for headers
- Body: "Inter" for data values
- Monospace: "JetBrains Mono" for numerical odds
